Allows you to swap specified loadouts by pressing a single button. (configuration included)

It's highly recommened to use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3017126912]Swap Loadouts While Using Items[/url] mod as well if you want to swap loadouts quickly.

There is no compatability with [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2990396828]Extended Equipment Loadouts[/url] mod.
